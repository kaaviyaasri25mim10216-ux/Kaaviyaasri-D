# Tic tac toe
## Overview:
Tic-Tac-Toe is a classic two-player strategy game implemented on a 3×3 grid. This project will implement a fully functional version of the above game, where two players will take turns to mark X and O on the board. The system checks for wins, draws, and invalid moves to ensure smooth gameplay.
This project is built as part of the Build Your Own Project evaluation. It follows all the guidelines regarding functionality, non-functionality, and documentation.

## How to Play
1. The game opens with a 3×3 grid with numbered positions.
2. Player X makes the first move, then Player O.
3. The players enter the position number to mark their symbol.
4. The system checks after every turn:
if any player wins
If the game had ended as a draw,
5. An end replay choice for the players.
